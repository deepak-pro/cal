# cal
A terminal based calender

## Installation

1. Clone this repository
 	
 	`git clone https://github.com/deepak-pro/cal && cd cal`
 
1. Compile using gcc

	`gcc calcu.c -o caclu`


## Usage 

*  Run `./caclu`  
*  To specify month and year run `./calcu s` then enter the month and year seperated by space in the prompt.
